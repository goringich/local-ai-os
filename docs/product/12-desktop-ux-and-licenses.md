# Desktop UX and license boundary

LOCAL AI OS integrates with the workstation's existing Hyprland and Rofi shell through one optional entry in the current `Super+D` launcher. It does not ship a second desktop shell, dashboard, compositor or software-rendered GUI path.

## Operator flow

- Smart Context: bounded Codex continuation pack.
- Find Knowledge: canonical-source retrieval with small snippets.
- What Now: goals and mission control.
- AI Control Center: existing operating view.
- Token Savings: measured context and retrieval quality.
- System Status: existing status command.
- Disable AI Navigator: non-destructive rollback to the previous launcher contents.

The launcher state is a sanitized runtime preference. Disabling the feature does not delete the control plane or change Hyprland visuals.
