# FAQ

LOCAL AI OS is a scoped implementation and operating contour for one private Linux workflow, not a general-purpose autonomous agent product.

## What can I evaluate publicly?

You can inspect the product site, proof contract, founder proofs, technical portal, release checks, static deployment configuration, and public documentation. Customer configurations, runtime state, source, prompts, and secrets are not public.

## Does cloning the repository install the system?

No. It builds the public product surface. Runtime and customer configuration are selected, reviewed, installed, and accepted in the delivery scope.

## Are metrics guaranteed?

No. A proof shows only recorded measurements. Missing measurements are explicitly unavailable. No customer result is implied by a founder proof.
