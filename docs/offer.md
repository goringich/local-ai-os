# LOCAL AI OS — pilot contract

## Positioning

LOCAL AI OS is an implementation service, not a new general-purpose agent runtime. It assembles a client's existing and selected local-first components into one documented, observable and recoverable Linux workspace.

## Pilot

- Price: 49,900 ₽
- Duration: five working days
- Scope: one Linux workstation and up to three projects
- Diagnostic: 9,900 ₽, fully credited toward the pilot

## Acceptance

The handoff includes checks for declared files, known promotion debt, secret risk, restore dry-run, UI/runtime behavior and a written report.

## Boundaries

- no full-autonomy claim;
- no raw prompt or response mirroring;
- no secret-bearing exports;
- no expansion of the client's trust boundary without explicit approval.
